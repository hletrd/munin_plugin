## MUNIN plugins

### ```cpu_per_user```
* Display CPU usage per user (unix style, 100% as full utilization of a single core)

### ```gpu_per_user_```
* Displays GPU usage per user

#### ```gpu_per_user_usage```
* GPU load per user

#### ```gpu_per_user_power```
* GPU power draw per user

#### ```gpu_per_user_memory```
* GPU memory per user
